## 🚀 Flagship Project — Nebula

> **Autonomous Drone Surveillance System** · ROS 2 · PX4 · YOLOv8 · Gazebo

Real-time threat-detection UAV (X500 airframe) with fine-tuned **YOLOv8s** and a 45° gimbal camera — detecting physical altercations in dynamic outdoor scenes. Custom **ThreatDetector** and **nebula_mission_controller** ROS 2 nodes enable fully autonomous pursuit from detection to engagement with zero manual input.

```
Sensors ──► YOLOv8s Detection ──► ThreatDetector Node
                                         │
                              nebula_mission_controller
                                         │
                         Visual Servoing + Altitude Control
                                         │
                              MAVROS ──► PX4 Autopilot
                                         │
                                   X500 Airframe
```
