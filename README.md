## Nebula — Autonomous Drone Surveillance System

> `ROS 2 Humble` · `PX4 Autopilot` · `YOLOv8s` · `Gazebo` · `X500 Airframe` · `C++ / Python`

An autonomous threat-detection UAV built end-to-end — from perception to mission execution. A fine-tuned **YOLOv8s** model with a 45° gimbal camera identifies physical altercations in dynamic outdoor scenes. Two custom ROS 2 nodes handle the complete mission loop:

- **`ThreatDetector`** — real-time object detection, classification, and threat-level assessment
- **`nebula_mission_controller`** — closed-loop visual servoing, adaptive altitude control, and autonomous pursuit

Zero manual input required from detection to engagement.

```
Camera Feed ──► ThreatDetector Node (YOLOv8s)
                       │
               Threat Confirmed?
                       │ Yes
               nebula_mission_controller
               ┌───────┴────────┐
        Visual Servoing   Altitude Control
               └───────┬────────┘
                    MAVROS
                       │
               PX4 Autopilot
                       │
             X500 Airframe Motors
```
