## Nebula — Autonomous Drone Surveillance

> Real-time threat detection · X500 airframe · ROS 2 · PX4 · YOLOv8s

Fine-tuned YOLOv8s with a 45° gimbal camera detects physical altercations in dynamic outdoor scenes. Custom `ThreatDetector` and `nebula_mission_controller` nodes handle closed-loop visual servoing, adaptive altitude control, and spotlight activation — zero manual input required.
