## About

Final-year **Aeronautical & Aerospace Engineering** student at New Mansoura University, Egypt — specializing in autonomous UAV systems, flight control, and high-fidelity aerodynamic simulation.

I build **production-grade autonomous drone systems** end-to-end — integrating ROS 2, PX4 Autopilot, computer vision, and 3D SLAM into real hardware. ## Nebula — Autonomous Drone Surveillance System

<div align="center">

> `ROS 2 Humble` &nbsp;·&nbsp; `PX4 Autopilot` &nbsp;·&nbsp; `YOLOv8s` &nbsp;·&nbsp; `Gazebo` &nbsp;·&nbsp; `X500 Airframe` &nbsp;·&nbsp; `C++ / Python`

</div>

An autonomous threat-detection UAV built end-to-end — from perception to mission execution. A fine-tuned **YOLOv8s** model with a 45° gimbal camera identifies physical altercations in dynamic outdoor scenes. Two custom ROS 2 nodes handle the complete mission loop with zero manual input:

- **`ThreatDetector`** — real-time detection, classification, and threat-level assessment
- **`nebula_mission_controller`** — closed-loop visual servoing, adaptive altitude control, and autonomous pursuit

```
Camera Feed ──► ThreatDetector Node (YOLOv8s)
                       │
               Threat Confirmed?
                       │ Yes
               nebula_mission_controller
               ┌───────┴────────┐
        Visual Servoing   Altitude Control
               └───────┬────────┘
                    MAVROS
                       │
               PX4 Autopilot
                       │
             X500 Airframe Motors
```
